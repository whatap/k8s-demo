<부하 테스트기>

각각 다른 요청을 보내는 버튼을 클릭해 애플리케이션의 부하를 발생시키고 이를 모니터링하여 원인을 찾아 봅시다!

헬스 체크
로그 생성

CPU 부하 상승
헬스 체크
caller 애플리케이션이 callee 애플리케이션의 상태를 체크하는 api 를 호출합니다.
로그 생성
caller 애플리케이션이 callee 애플리케이션을, callee 애플리케이션이 callee-with-db 애플리케이션의 log 생성 api 를 호출합니다.
DB 느리게
caller → callee → callee-with-db 순으로 호출하며 callee-with-db 에서 트랜젝션이 매우 느리게 진행됩니다.
CPU 부하 상승
caller → callee 순으로 호출하며 callee 에서 무거운 연산이 오랫동안 진행됩니다.
